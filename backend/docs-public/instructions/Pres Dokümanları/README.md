# Pres Dokümanları

Bu klasöre bakım ve teknik dokümantasyonlarınızı ekleyin.

## Desteklenen Formatlar

- **PDF** (.pdf)
- **Word** (.docx, .doc)
- **Metin** (.txt)
- **Markdown** (.md)

## Nasıl Kullanılır?

1. Dokümanlarınızı bu klasöre kopyalayın
2. Backend sunucusunu başlatın: `cd backend && npm run dev`
3. Chat arayüzünden sorularınızı sorun: `http://localhost:5173/chat`

## Örnek Sorular

- "yv90 nedir?"
- "400 hata kodu nedir, nasıl çözülür?"
- "Makine sıcaklığı nedir?"
- "Test aşamalarını birbir anlatır mısın?"

## Notlar

- Sistem bu klasördeki tüm desteklenen dosyaları otomatik olarak okur
- Dokümanlar değiştiğinde sunucuyu yeniden başlatmanız gerekmez (her soruda yeniden okunur)
- Maksimum dosya boyutu: 50MB
